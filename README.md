# Fora9 Agency - Scholarships Project

A web project for managing educational scholarships, allowing students to apply and track scholarships through a dashboard for **students** and **admins**.

---

## Features
- Secure login for Admin and Student
- Add, edit, and view scholarships
- Submit applications and track status
- User-friendly dashboard

---

## Technologies
- PHP + MySQL (PDO)
- HTML / CSS / JavaScript
- XAMPP / any local server
- GitHub for version control
